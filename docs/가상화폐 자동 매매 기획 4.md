v5 의 핵심 변경:

1. **단일 엔진** — 8개 엔진을 폐기하고 `TradingEngine` 1개로 통합
2. **WebSocket 우선** — 폴링을 버리고 Upbit `wss://api.upbit.com/websocket/v1` 로 시세 수신
3. **Regime-Switching** — Trend / Range / Chaos 분류 후 전략 자동 전환
4. **Risk-First 사이징** — 트레이드당 자본의 1% 손실로 포지션 크기 역산
5. **Paper / Live Shadow** — 두 PnL 곡선을 동시 측정해 slippage cost 정량화
6. **ML/LLM 강등** — 가격 예측에서 제외, 보조 필터로만 잔존
