## 1. 프로젝트 개요

### 1.1 프로젝트 명

> “AutoTraderX” — 업비트 API 기반 실시간 코인 단기 자동 매매 시스템
> 

### 1.2 목적

- **실시간 시세 데이터를 기반으로 단기 매매 자동화**
- **트레이더(워뇨띠) 스타일의 리스크 최소화형 단타 전략 적용**
- **인공지능 기반 매매 판단(패턴 인식, 감정 배제)**
- **사용자 설정 가능한 웹앱 인터페이스 제공 (종목, 임계값, 수익률 목표)**

### 1.3 주요 특징

- 업비트 **오픈 API 거래 자동화** ([https://upbit.com/service_center/open_api_guide](https://upbit.com/service_center/open_api_guide))
- 실시간 **Tick 단위 시세 분석**
- **AI 기반 매매 판단 로직** (상승 전환 감지 / 패턴 리스크 회피)
- **웹 기반 상태 모니터링 시스템** (React + FastAPI)
- **손절/익절 자동 설정** 및 **API 보안 관리**

## 2. 벤치마킹 및 전략 분석

### 2.1 워뇨띠 투자 방식 요약 (단기 트레이딩 기반)

- **캔들 변화, 거래량 급등, MACD 골든크로스 등 단기 패턴 포착**
- **상승 추세 진입 시 적극 매수, 반전 조짐 시 즉시 매도**
- **절대 장기 보유 금지**
- **손절 라인 명확히 설정 (보통 -1% ~ -2%)**
- **익절 라인도 고정 (보통 +1% ~ +3%)**
- **리스크 최소화를 위해 분할 매매 또는 멀티 티커 소액 분산 운영**

### 2.2 자동 매매 전략 방향성

| 항목 | 전략 |
| --- | --- |
| 거래방식 | 단기 스캘핑 / 알고리즘 트레이딩 |
| 분석주기 | 1초~5초 단위 Tick 데이터 |
| 주요지표 | RSI, MACD, MFI, 이동평균선 |
| 매수조건 | 상승 추세 전환 지표 2개 이상 일치 시 |
| 매도조건 | 거래량 감소 + 수익률 목표 도달 or MACD 데드크로스 |
| 손절조건 | 급락 1% 이상 발생 or 패턴 이탈 감지 |

## 3. 시스템 구성도

### 3.1 전체 아키텍처

```
[사용자 브라우저]
      │
      ▼
[웹앱 (Frontend: React)]
      │
      ▼
[API 서버 (FastAPI / Flask)]
      │
      ├── [업비트 Open API 통신 모듈]
      │       └─ 시세조회 / 주문하기 / 주문조회 / 잔고조회
      │
      ├── [매매 알고리즘 엔진]
      │       ├─ 시세 데이터 버퍼 (Redis)
      │       ├─ 패턴 인식 모듈 (AI 모델 or 기술적 지표)
      │       ├─ 자동매수/매도 로직
      │
      ├── [Account 관리 & Risk Control]
      │       ├─ API Key 암호화 저장 (AES256)
      │       ├─ 주문 수 제한 / 슬리피지 처리
      │
      └── [DB: PostgreSQL / MongoDB]
              ├─ 거래 로그
              ├─ 시세 히스토리
              ├─ 백테스트 기록

```

## 6. 전략 자동화 로직 설계 예시

### 6.1 간단한 트레이딩 알고리즘 흐름

```python
def auto_trade_loop():
    ticker = "KRW-BTC"
    price = get_current_price(ticker)
    indicators = get_technical_indicators(ticker)

    if indicators["macd_cross"] == "golden" and indicators["rsi"]  80:
        sell_coin(ticker)

```

### 6.2 강화된 조건 판단 (워뇨띠식)

- 상승 초입 감지: **거래량 증가 + 단기 이동평균선 상향 돌파 + RSI 상승세**
- 매도 조건: **거래량 급감 + MACD 데드 크로스 + 수익률 +2% 이상 시 자동 청산**
- 안전장치: **최근 3분 내 변동폭 > 5% 시 신규 진입 금지 (리스크 회피)**
